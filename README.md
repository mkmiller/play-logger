# Playlogger

> You can't manage what you can't measure. </br>
> -- Author unknown

**Playlogger** is a tool to track your game plays and help you make decisions about your game collection.

## Setup

```
$ bundle install
$ rails server
```

## Dependencies
* Ruby 2.3.1
* Rails 5.0.0.1
* Postgres >= 9.3.5

## Tests
Tests are minitest and can be run with rake.

```
$ rails test
```
